1. Go into server.cfg, then add your database name to these lines:

	set mysql_connection_string "server=127.0.0.1;database=essentialmode;userid=root;password="

2. Go to the bottom and add your web-api-key:

	set steam_webApiKey "ovde-web-api" 

3. After that add your fivem-key:

	sv_licenseKey ovde-ubaci-key

Add the .sql fine, run the server for the first time, let it form and you're done. Enjoy!
